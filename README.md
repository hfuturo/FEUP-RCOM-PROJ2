# Instruções para correr programa

### Compilar:

```bash
$ make
```

### Correr programa com variáveis definidas pelo user

```bash
$ make user=? password=? host=? url=? run
```
, onde `?` são as variáveis introduzidas pelo user no terminal

### Correr testes

```bash
$ make run_teste
```

```bash
$ make run_testeX
```
X = (1 a 7)

### Limpar:

```bash
$ make clean
```
